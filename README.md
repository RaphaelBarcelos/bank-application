# Aplicação bancaria em java
Simulação de aplicação bancária desenvolvida em Java para praticar e demonstrar domínio dos fundamentos da linguagem — especialmente orientação a objetos e encapsulamento.

# Funcionalidades

Criação de contas
Transferências entre contas
Histórico de pagamentos

Toda a lógica roda em memória — sem banco de dados, sem framework. Java puro.

# Conceitos aplicados

Orientação a objetos — modelagem de entidades como Conta, Transacao e Cliente com responsabilidades bem definidas
Encapsulamento — controle de acesso aos dados internos de cada classe
Coleções — gerenciamento de contas e histórico de transações em memória

# Stack
```
Java — lógica de negócio e estrutura da aplicação
React — interface para interação com o sistema
```
